# 4.1 Vorarbeit: Container / Docker
Stelle sicher, dass du die folgenden Punkte verstanden hast:

- Ich kann erklären, welches Problem Docker löst.
- Ich kann den Unterschied zwischen einem Image und einem Container in eigenen Worten beschreiben.
- Ich weiss, was in einem Dockerfile steht und wie es aufgebaut ist.
- Ich verstehe, warum Container ihre Daten verlieren und was Volumes dagegen tun.
- Ich kann erklären, was eine Container Registry ist und wozu man sie braucht.
- Ich weiss, was Docker Compose ist und wann man es einsetzt.
- Ich kenne den Unterschied zwischen Docker Compose (lokal) und Container-Orchestrierung (Cloud, z.B. ECS).
